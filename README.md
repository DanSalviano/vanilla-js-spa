# vanilla-js-spa

### Run this project

To run this code, do the following:

1. Install Node.js
2. Clone the project
3. Navigate to your project folder and run the following in a terminal:
- `npm init -y` (to create a Node.js project)
- `npm i express` (to install Express)
- `npm run dev` (to run the server)
4. Open `localhost:3001` in a web browser (the current port specified in `server.js` is 3001. If you need to change it, go ahead)
